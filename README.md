---
title: User Flow - Система учета практики студентов
---
flowchart TD
    A[Начало] --> B{Есть аккаунт?}
    B -->|Нет| C[Регистрация<br>email, пароль, ФИО]
    B -->|Да| D[Вход<br>email + пароль]
    C --> D
    
    D --> E[Личный кабинет]
    E --> F{Выбор действия}
    
    F -->|Просмотр профиля| G[Редактирование профиля]
    F -->|Документы практики| H[Раздел документов]
    
    H --> I{Действие с документами}
    I -->|Создать новый| J[Заполнение формы документа]
    I -->|Просмотреть| K[Список документов]
    
    J --> L[Сохранение документа]
    L --> K
    
    K --> M{Действие}
    M -->|Редактировать| J
    M -->|Удалить| N[Удаление документа]
    M -->|Скачать| O[Экспорт документа]
    M -->|Назад| H
    
    G --> E
    N --> K
    O --> K
    
    H --> P[Выход из системы]
    E --> P
    
    P --> Q[Конец сеанса]
